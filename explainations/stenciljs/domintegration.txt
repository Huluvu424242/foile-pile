DOM Integration

Auf dieser Folie sehen Sie wie eine Webkomponente als <lang xml:lang="en-US">TAG</lang> im DOM eingebaut wurde.
Hierbei wird zwischen dem <lang xml:lang="string">Light</lang>DOM im Browser und dem <lang xml:lang="en-US">Shadow</lang>DOM in der Webkomponente unterschieden.

Der <lang xml:lang="string">Light</lang>DOM des Browsers kann von der Webseite per CSS stylistisch bestimmt und verändert werden.
Auch das Host <lang xml:lang="en-US">TAG</lang> der Webkomponente lässt sich noch per CSS von der Webseite stylen.

Der<lang xml:lang="en-US">Shadow</lang>DOM der Webkomponente kann nur von der Webkomponente selbst stylistisch bearbeitet werden.
Ein Styling von außerhalb der Webkomponente wird vom Browser verhindert.
